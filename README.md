## Cara Menjalankan Lokal

1. Clone repository ini.
2. Buka folder project.
3. Jalankan file `index.html` langsung di browser.

## Cara Deploy GitHub Pages

1. Push seluruh file project ke branch `main`.
2. Masuk ke repository GitHub.
3. Buka `Settings > Pages`.
4. Pastikan source menggunakan `GitHub Actions`.
5. Workflow deploy akan berjalan otomatis dan mengunggah website ke GitHub Pages.

## Penjelasan CI/CD

Push ke branch `main` akan menjalankan GitHub Actions dan melakukan deploy otomatis ke GitHub Pages. Workflow akan checkout repository, mengonfigurasi Pages, mengunggah artifact website statis, lalu melakukan deploy ke environment GitHub Pages.
